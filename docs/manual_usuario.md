Inicio de Sesion:
    Para iniciar sesión se coloca el email del usuario y la contraseña registrada. En el caso de que no tenga una cuenta, entonces el usuario presiona en donde dice "Create New Account" para crear una nueva.

Crear ticket:
    Una vez iniciado sesion en la cuenta, en el dashboard aparecerá una opción que dice "New Ticket", y dentro de él colocas el título del ticket junto con su descripción.

Ver tickets:
    Para ver tus tickets, vas a la sección de 'tickets' arriba en el navbar, el cual te llevará a la lista de todos tus tickets. Si deseas verlo más detalladamente presionas el botón 'see' que se encuentra en el ticket.

Como ve un Admin vs un Agent vs un User:
    Admin:
        El admin puede ver la lista de usuarios que se encuentran registrados en la página, y a su vez, puede asignar lo roles que desee a cada usuario, sin embargo, el admin no puede crear tickets.

        Además, puede asignarle un ticket a alguno de los agentes para que los trabajen, y cambiar el estado del ticket.
    Agent:
        El agent NO puede de igual forma crear tickets, y tampoco puede ver la lista de usuarios. 
        Ellos se encargan de manejar los tickets asignados a ellos.
    User:
        El usuario es quien crea los tickets. Solamente puede crearlos y ver su estado.
